# WebSocket Compose Desktop

## Project starter

### Client

* Kotlin 1.4.20
* Spring 5.2.12
* Jackson 2.11.3

#### Running the client
```
cd websocket-client-compose-desktop
gradle :run
```


### Server

* Kotlin 1.4.32
* Spring 5.2.12
* Jackson 2.11.3
* Tomcat 8.5.12

#### Running the service
```
cd websocket-service
build.cmd
cd bin
service.cmd
```
